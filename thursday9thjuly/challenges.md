# Challenges

## Git and GitHub

**Status:** Solved

I initially found Git and GitHub challenging, particularly when cloning repositories, pulling changes between different computers, resolving file path issues and pushing updates correctly.

To solve these issues, I:

- Checked that I was in the correct directory.
- Used `git status` to check the state of the repository.
- Checked that the remote repository was set correctly.
- Practised using the main Git commands regularly.

Some of the commands I used were:

```bash
git status
git add .
git commit -m "Update notes"
git pull origin main
git push origin main
```

Practising these commands throughout the course has helped me become more confident with Git and GitHub.
